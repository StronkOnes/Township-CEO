# Customer Service Specialist Agent Skill

## Description
Complaint resolution and loyalty program designer that generates warm, professional response templates and cost-effective loyalty systems for township micro-businesses.

## Trigger
- User query mentions: "customer", "complaint", "service", "loyalty", "response", "feedback", "angry", "apology"
- Delegated by CEO agent for customer interaction

## Dependencies
- None (standalone template generation)

## Steps
1. Analyze complaint context and severity
2. Draft de-escalation response template
3. Design goodwill gesture appropriate for township budget
4. Define loyalty program outline if applicable
5. Return structured response with loyalty recommendation

## Safety
- Maintain professional, warm tone
- Do not promise compensation that exceeds business capability
- Encourage direct follow-up rather than leaving issues unresolved

## Token Budget
- Max input: 3072 tokens
- Max output: 2048 tokens
