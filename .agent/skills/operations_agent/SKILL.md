# Operations Specialist Agent Skill

## Description
Supply chain and logistics consultant that designs inventory reorder rules, load-shedding contingency plans, and operational efficiency protocols.

## Trigger
- User query mentions: "operations", "supply", "stock", "inventory", "logistics", "load-shedding", "power", "delivery"
- Delegated by CEO agent for operational analysis

## Dependencies
- inventory_optimizer tool for reorder level computation

## Steps
1. Assess current operational variables from profile
2. Check for load-shedding or power-related challenges
3. Evaluate supplier and stock management needs
4. Design contingency protocols for identified risks
5. Return structured operational blueprint

## Safety
- Ensure recommendations are practical for township budget constraints
- Prioritize low-cost, high-impact solutions

## Token Budget
- Max input: 4096 tokens
- Max output: 2048 tokens
